# Data Model: textbook-generation

**Feature Branch**: `001-textbook-gen`
**Created**: 2025-12-05
**Spec**: [specs/001-textbook-gen/spec.md](specs/001-textbook-gen/spec.md)

## Entities

### Textbook Chapter
Represents a discrete unit of content within the textbook.

-   **id**: Unique identifier for the chapter (e.g., UUID, slug).
-   **title**: Display title of the chapter (e.g., "Introduction to Physical AI").
-   **content**: The full structured content of the chapter, including text, images, and code examples (e.g., Markdown format).
-   **order**: Numerical value to determine the display order of chapters.

### User Query
Represents a question or input provided by the user to the RAG chatbot.

-   **text**: The actual text of the user's question.
-   **timestamp**: When the query was made.
-   **context_text**: (Optional) The selected text from the Docusaurus UI that provides context for the query.

### RAG Response
Represents the answer generated by the RAG chatbot based on the textbook content.

-   **response_text**: The synthesized answer provided by the chatbot.
-   **source_chapters**: A list of `Textbook Chapter` IDs or references from which the answer was derived.
-   **timestamp**: When the response was generated.

### Embedding
Represents the numerical vector embedding of a piece of text content from the textbook.

-   **id**: Unique identifier for the embedding (e.g., hash of content, UUID).
-   **vector**: The numerical vector representation of the text content.
-   **text_chunk**: The original text segment from which the embedding was generated.
-   **chapter_id**: Reference to the `Textbook Chapter` this embedding belongs to.
